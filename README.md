# TeenTaal
Virtual Machine for the TeenTaal.
In Dutch, Teen means toe and Taal means language.
So, TeenTaal means ToeLanguage.

# Building
Requirements: 
* GCC
Go to the OS folder to see the supported build methods for your OS
